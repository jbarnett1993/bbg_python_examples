Bloomberg Terminal API  SDK

 

Requirements

·        Operating System: Windows XP SP3 (32bit), or Windows Vista (32 or 64 bit), or Windows 7 (32 or 64 bit).

·        Microsoft .Net 4.0 Client Profile or latter.

 

Terminal API SDK includes

·        Bloomberglp.TerminalApiEx.dll version 1.2.8.3 incorporating functionality to enable third party applications to interact with Bloomberg Terminal.

·        Bloomberglp.Component.Wpf.UI.dll version 1.2.8.3 incorporating functionality to enable third party WPF applications to embed LaunchPad components.

·        Bloomberglp.Component.Forms.UI.dll version 1.2.8.3 incorporating functionality to enable third party WinForm applications to embed LaunchPad components.

 

Directory Structure

By default Terminal SDK is installed in C:\blp\TerminalApiSdk folder.

·        Assemblies are located in .\lib subfolder.

·        Samples are located in .\Samples subfolder.

·        Help file is located in .\Help subfolder.
